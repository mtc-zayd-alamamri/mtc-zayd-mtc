# UML Tool Evaluation

| Criterion | Tool A: diagrams.net | Tool B: PlantUML Web |
|---------|----------------------|----------------------|
| Learning curve | Very easy, drag-and-drop interface | Requires UML syntax, fast after learning basics |
| Version control friendliness | .drawio can be diffed as XML; PNG for review | .puml is text-diff friendly; PNG for review |
| Validation support | Manual visual checks only | Syntax validation and basic lint via editor |
| Collaboration | Works with Google Drive and OneDrive | Text is PR-friendly; code reviews are easy |
| Export formats | PNG, SVG, PDF | PNG (can regenerate from source) |
| Cost | Free | Free |
